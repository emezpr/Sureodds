
# SureOdds AI - Professional Football Prediction Engine

This is a high-performance football analysis engine that uses the Google Gemini 3 Pro API to provide the 5 most statistically certain betting predictions daily.

   - **Vercel/Netlify**: Add `API_KEY` in the Environment Variables section of your project settings.

2. **Run Locally**:
   If you have Node.js installed:
   ```bash

   ```
